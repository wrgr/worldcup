Model and source notes
Prediction layer: each unresolved match uses the embedded ELO table and the standard logistic transform 1/(1+10^(-ΔELO/400)). It ignores injuries, rest, travel, home advantage, styles, and penalty-specific skill unless already reflected in the rating.

Live layer: the page first reads scoreboard.json committed to the repo root. If that file is missing or stale, it tries ESPN's public World Cup scoreboard feed directly in the browser. If both fetches fail, it falls back to the embedded knockout snapshot. Official fixtures/results should be checked against FIFA's fixture page.
